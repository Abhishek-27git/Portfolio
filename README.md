## 📖 About The Project

This is my personal portfolio website designed to showcase my projects, skills, education, and professional journey. It features a clean, highly symmetrical, grid-based editorial layout. Built on the cutting edge of web-development tools, it offers a fast, accessible, and interactive experience.

## ✨ Key Features

- **Editorial Layout**: A premium, grid-based aesthetic inspired by modern magazine design.
- **Dark Mode First**: Fully implemented and responsive dark mode using `next-themes`.
- **Smooth Animations**: High-performance entry, hover, and scroll animations built with **Framer Motion**.
- **Smooth Scrolling**: Implemented using **Lenis** for a seamless, continuous scrolling experience.
- **Accessible UI**: Uses accessible primitives provided by **Radix UI** (Tabs, Dialogs, Hover Cards, etc).
- **Responsive Design**: Tailored layout for all screen sizes from mobile to desktop.
- **Performance Optimized**: Scoring high on Core Web Vitals, fully cached and optimized by Next.js.
- **Analytics Ready**: Integrated with **Vercel Analytics** and **Speed Insights**.

## 🚀 Tech Stack

### Core Frameworks
- **[Next.js 15](https://nextjs.org/)** (App Router)
- **[React 19](https://react.dev/)**
- **[TypeScript](https://www.typescriptlang.org/)**

### Styling & UI
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first styling.
- **[Framer Motion](https://www.framer.com/motion/)** - Declarative animations.
- **[Radix UI](https://www.radix-ui.com/)** - Unstyled, accessible UI components.
- **[Lucide React](https://lucide.dev/) & [Simple Icons](https://simpleicons.org/)** - Beautiful, consistent iconography.

### State & Utilities
- **[Lenis](https://lenis.studiofreight.com/)** - Smooth scrolling.
- **[Sonner](https://sonner.emilkowal.ski/)** - Opinionated toast components.
- **[Embla Carousel](https://www.embla-carousel.com/)** - Lightweight carousels.

## 📂 Project Structure

```text
├── app/               # Next.js App Router (pages and layouts)

```

## 🛠️ Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed along with `npm` (or `yarn`, `pnpm`, `bun`).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Abhishek-27git/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file in the root directory and add any necessary environment variables (like Google Analytics ID or NodeMailer configs).

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open the app:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
